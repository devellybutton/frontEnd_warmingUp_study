## 기능 구현
- JavaScript를 활용하여 DOM 조작 방식으로 구현:
  - `createMenuItem` 함수: HTML 요소 생성
  - `filterMenuItems` 함수: 메뉴 항목 필터링
  - `insertMenuItem` 함수: 필터링된 항목을 DOM에 삽입
