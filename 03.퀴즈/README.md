## 메인 기능
- 4가지 퀴즈 유형을 선택할 수 있는 메인 화면 구현
  - OX 퀴즈 (✅)
  - 초성 퀴즈 (⏳)
  - 사칙연산 퀴즈 (⏳)
  - 가로세로 낱말퍼즐 (⏳)
- '처음으로' 버튼으로 언제든지 메인 화면으로 복귀 가능

## OX 퀴즈
1. 랜덤으로 10개 문제 생성
2. 사용자 응답(O 또는 X) 후 정답 여부 표시 및 해설 제공
- 2초 후 자동으로 다음 문제로 넘어감
3. 현재 문제 번호와 총 문제 수 표시 (예: "1 / 10")
- 맞힌 문제 수 실시간 카운트
4. 특정 문제 스킵도 가능
5. 10문제 완료 후 자동으로 메인 화면으로 복귀
